7. Difference between loc and iloc
iloc (Integer Location)
Position based indexing
s.iloc[2]
Returns value at position 2

loc
Label based indexing
Example
s = pd.Series(
    [25,30,22,28,27],
    index=["Aman","Ravi","Priya","Neha","Rahul"]
)
s.loc["Priya"]
22

Important Difference
Method Type
iloc  Position based
loc   Label based
Slice difference
iloc[1:4] -> stop NOT included
loc["Ravi":"Neha"] -> stop INCLUDED

Pandas Joins & Merge
Example DataFrames
import pandas as pd

Inner Join
Keep only matching rows from both DataFrames
pd.merge(df1, df2, on="id", how="inner")
Output
id  name  salary
1  Aman   10000
2   Raj   15000
Left Join
Keep all rows from df1 and matching rows from df2
pd.merge(df1, df2, on="id", how="left")
Output
id  name  salary
1  Aman  10000
2  Raj   15000
3  Ravi   NaN

Right Join
Keep all rows from df2 and matching rows from df1
pd.merge(df1, df2, on="id", how="right")
Output
id  name  salary
1  Aman  10000
2   Raj  15000
4   NaN  20000

Outer Join
Keep all rows from both DataFrames
pd.merge(df1, df2, on="id", how="outer")
Output
id  name  salary
1  Aman   10000
2  Raj    15000
3  Ravi    NaN
4  NaN    20000

Cross Join
Every row of df1 joins with every row of df2
pd.merge(df1, df2, how="cross")
Example result
name salary

Aman  20000
Raj  10000
Raj  15000
Raj  20000

Merge vs Join
merge()
Used for column-based join
pd.merge(df1, df2, on="id")
join()
Used for index-based join
df1.join(df2, how="inner")

What is chunk in Pandas?
A chunk means reading a large file in small parts (batches) instead of loading the full file at once into memory.
Imagine a CSV has 10 million rows.
Without chunk:
Python
df = pd.read_csv('large_file.csv')
➡ Entire file loads into RAM at once.
➡ Can become slow or give memory error.
With chunk:
Python
for chunk in pd.read_csv(
    'large_file.csv',
    chunksize=10000
):
    print(chunk.head())
Here:
chunksize=10000
means:
Read 10,000 rows at a time
Example
CSV has 25,000 rows
If:
Python
chunksize=10000
Then Pandas creates:
Chunk 1: 10,000 rows
Chunk 2: 10,000 rows
Chunk 3: 5,000 rows
Why use chunk?
Large file handling
Prevent memory issue
Faster processing for huge datasets
Process data step by step
