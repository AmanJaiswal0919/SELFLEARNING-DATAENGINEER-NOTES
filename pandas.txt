1. What is Pandas? Why do we need Pandas?
Answer:
Pandas is a Python library used for data analysis and manipulation.
It provides data structures like Series and DataFrame to work with structured data.
Without Pandas:
Data in NumPy arrays has no column labels
Hard to read and analyze large datasets
With Pandas:
Data is stored in tabular format (like Excel)
Easy to filter, sort, clean, and analyze data
Example use:

2. What are the Primary Data Structures in Pandas?
Pandas has two main data structures
Series
1-Dimensional
Homogeneous data
Mutable
Like a single column in a table
Example
s = pd.Series([10,20,30,40])
print(s)
Output
0    10
1    20
2    30
3    40
dtype: int64

DataFrame
2-Dimensional
Heterogeneous data
Like an Excel table
Example
df = pd.DataFrame({
    "Name":["Aman","Ravi"],
    "Age":[25,30]
})

7. Difference between loc and iloc
iloc (Integer Location)
Position based indexing
s.iloc[2]
Returns value at position 2

loc
Label based indexing
Example
s = pd.Series(
    [25,30,22,28,27],
    index=["Aman","Ravi","Priya","Neha","Rahul"]
)
s.loc["Priya"]
22

Important Difference
Method Type
iloc  Position based
loc   Label based
Slice difference
iloc[1:4] -> stop NOT included
loc["Ravi":"Neha"] -> stop INCLUDED

9. DataFrame Example
import numpy as np

data = {
"Name":["Alice","Bob","Charlie","David","Eve","Alice"],
"Age":[25,30,35,np.nan,29,25],
"Department":["HR","IT","Finance","IT","HR","HR"],
"Salary":[50000,60000,70000,62000,np.nan,50000]
}

df = pd.DataFrame(data)
Output
Name  Age  Department  Salary
Alice 25     HR      50000
Bob   30     IT      60000
Charlie 35  Finance   70000

Pandas DataFrame – Important Operations

Viewing Data
head()
Shows first rows
df.head(2)
Output → first 2 rows
tail()
Shows last rows.
df.tail(3)
Output → last 3 rows

Row & Column Selection
Using iloc (position based)
df.iloc[1:3]
Returns rows 1 and 2
Select rows and columns
df.iloc[1:3, ["Age","Department"]]

Dropping Columns or Rows
Drop Column
df.drop("Age", axis=1)
axis=1 → column
Drop Row
df.drop(2, axis=0)
axis=0 → row
Important Note
By default dataframe does NOT change
df.drop("Age", axis=1)
To change original dataframe
df.drop("Age", axis=1, inplace=True)


DataFrame Information
Shape
(rows, columns)
(6,4)
df.info()
Shows
column names
non-null values
data types
Example
Name           object
Age            float64
Department     object
Salary         float64

Statistical Summary
df.describe()
Works on numeric columns
Output
count
mean
std
min
max

Broadcasting
Apply operation to entire column
Example
df["Salary"] = df["Salary"] + 5000
All salary values increase by 5000

Renaming Columns
df.rename(columns={"Department":"Dept"}, inplace=True)

Unique Values
df["Dept"].unique()
Output
['HR','IT','Finance']

Value Counts
df["Dept"].value_counts()
Output
HR         3
IT         2
Finance    1

Creating New Column
df["PromSalary"] = df["Salary"] * 10

Handling Missing Values
Count missing values
df.isnull().sum()
Example
Name       0
Age        1
Dept       0
Salary     1

Dropping Missing Values
Drop rows with any NULL
df.dropna() or df.dropna(how="any")
Drop rows where ALL values are NULL
df.dropna(how="all")

Fill Missing Values
Fill with constant
df.fillna(0)
Fill with mean
df["Age"].fillna(df["Age"].mean())
Forward Fill
df["Age"].fillna(method="ffill")
Replace null with previous value
Backward Fill
df["Age"].fillna(method="bfill")
Replace null with next value


If first value is null, ffill will not replace it
If last value is null, bfill will not replace it
Replace Values
df["Name"] = df["Name"].replace("Charlie","Rose")

Duplicate Values
Find duplicates
df[df.duplicated()]
Keep last duplicate
df[df.duplicated(keep="last")]
Remove duplicates
df.drop_duplicates()

Lambda Function
Example
df["Promoted Salary"] = df["Promoted Salary"].apply(
    lambda x: x/10 if x>650000 else x
)
Apply Function
def multiplying_age(x):
    return x*2
df["Age"] = df["Age"].apply(multiplying_age)

Joins / Merge
df1 = pd.DataFrame({
"id":[1,2,3],
"name":["Aman","Raj","Ravi"]
})

df2 = pd.DataFrame({
"id":[1,2,4],
"salary":[10000,15000,20000]
})


Inner Join
pd.merge(df1, df2, on="id", how="inner")
Only matching rows

Left Join
pd.merge(df1, df2, on="id", how="left")
All rows from left dataframe

Pandas Joins & Merge
Example DataFrames
import pandas as pd

Inner Join
Keep only matching rows from both DataFrames
pd.merge(df1, df2, on="id", how="inner")
Output
id  name  salary
1  Aman   10000
2   Raj   15000
Left Join
Keep all rows from df1 and matching rows from df2
pd.merge(df1, df2, on="id", how="left")
Output
id  name  salary
1  Aman  10000
2  Raj   15000
3  Ravi   NaN

Right Join
Keep all rows from df2 and matching rows from df1
pd.merge(df1, df2, on="id", how="right")
Output
id  name  salary
1  Aman  10000
2   Raj  15000
4   NaN  20000

Outer Join
Keep all rows from both DataFrames
pd.merge(df1, df2, on="id", how="outer")
Output
id  name  salary
1  Aman   10000
2  Raj    15000
3  Ravi    NaN
4  NaN    20000

Cross Join
Every row of df1 joins with every row of df2
pd.merge(df1, df2, how="cross")
Example result
name salary
Aman  10000
Aman  15000
Aman  20000
Raj  10000
Raj  15000
Raj  20000

Merge vs Join
merge()
Used for column-based join
pd.merge(df1, df2, on="id")
join()
Used for index-based join
df1.join(df2, how="inner")
