2) GROUP BY AGGREGATE FUNCTION
you have to find out unique value with aggregation use GROUP BY AGG FUNCTION 
example-average salary per department empid-1,2,3 dept-IT, IT, HR salary-5000, 7000, 4000
SELECT dept, AVG(salary)
FROM EMPLOYEES
GROUP BY dept; GROUP1-IT, IT GROUP2-HR after this aggregation 
applied on GROUP1-IT, IT (5000+7000)/2 =6000 GROUP2-HR 4000 (4000)/1=4000 
output- IT 6000, HR 4000

3) DISTINCT+GROUP BY
example-count unique product sold per date 
date- 1Jan, 1Jan, 1Jan product-A,A,B 
#at the time of display COUNT for GROUP1-2(A,A), for GROUP2-1(B)
SELECT date, COUNT (product) FROM SALES
GROUP BY date, product; GROUP1-13an A, IJAN A GROUP2-1JAN-B 
output-1Jan 2,1Jan 1

#at the time of display DISTINCT-A,B COUNT=2 so our output will be 1Jan 2 
SELECT date, COUNT(DISTINCT product) FROM SALES
GROUP BY date; GROUP1-1Jan
output-1Jan 2


4) Aggregate function without GROUP By
you have to find just single aggregate value from a column not per column category
-find the total number of employees in the company-Only Aggregate (here we need one single result from whole table)
SELECT COUNT(*)
FROM Employees
-find the number of employees in each department-GROUP BY+Aggregate 
(here we need category (department) wise result) 
SELECT dept, COUNT(*)
FROM Employees
GROUP BY dept;

7) LIMIT-OFFSET
-Nth Highest value, Top N records
-Find second highest salary?
SELECT salary
FROM Employees
ORDER BY salary DESC
LIMIT 1 OFFSET 1;
-Find Top 3 highest salaries
SELECT salary
FROM Employees
ORDER BY salary DESC
LIMIT 3;

ques-show all customers with their orders?

ques-find customers who never order?
